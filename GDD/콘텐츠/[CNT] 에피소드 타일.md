## 상세 사양(병합 본문)

## 목적
- 에피소드 타일에서 간단한 대화를 통해 상태 변화와 분기를 유도한다.

## 흐름
1) 노드 진입 → 대화 시작  
2) 1~3개의 선택지 제시(선택지 없는 단일 내러티브도 가능)  
3) 선택에 따른 효과 적용(플래그/호감도/아이템/해금)  
4) 다음 노드로 이동

## UI
- 상단: 배경/장면명  
- 좌/우: 화자 초상/이름  
- 중앙: 대사 텍스트, 하단 선택지 버튼  
- 스킵/로그 보기/자동진행 옵션

## 데이터 예시
```json
{
  "episodeId": "EP-021",
  "scene": [
    {"speaker": "동료A", "text": "준비는 됐어?"},
    {"speaker": "주인공", "text": "가자."}
  ],
  "choices": [
    {
      "label": "바로 떠난다",
      "conditions": [],
      "effects": [{"type": "flag.set", "key": "ready", "value": true}],
      "next": "N-201"
    },
    {
      "label": "보급을 확인한다",
      "conditions": [],
      "effects": [{"type": "provision.add", "item": "응급약", "value": 1}],
      "next": "N-105"
    }
  ]
}
```

## 검증 항목
- 선택지 텍스트와 실제 효과 일치  
- 잠금 선택지는 비활성/라벨에 사유 표시  
- 로그와 되돌리기(선택 직전까지) 옵션
